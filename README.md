# Worktime

A simple time tracking app for your projects.

> [!WARNING]
> This project is still under heavy development and a lot of features are missing.

## Stack

- SvelteKit
- Typescript
- DrizzleOrm
- Tailwind CSS
- shadcn-svelte
- Coolify CD/CI
